# Recomendador-de-Equipamentos-de-Rede-ConectaJ-
🚀 Recomendador de Equipamentos (PHP)
Um sistema simples que sugere equipamentos de rede (Roteadores/Switches) de acordo com o ambiente selecionado.

🛠️ O que usei:
HTML5 (Estrutura)
PHP (Lógica com switch-case)
XAMPP (Servidor Local)

📂 Como rodar:
Coloque os arquivos na pasta htdocs do seu XAMPP.
Inicie o Apache.
Acesse http://localhost/nome-da-sua-pasta/ no navegador.

🧠 O que aprendi:
Manipulação de formulários (POST).
Segurança básica com htmlspecialchars.
